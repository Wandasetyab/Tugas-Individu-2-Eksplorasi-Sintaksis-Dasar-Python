## Cakupan Praktikum
Tahapan koding yang dilakukan mencakup:
1.  **Mekanisme I/O Terintegrasi:** Implementasi pengambilan data pengguna melalui fungsi `input()` dan penyajian informasi menggunakan `print()` dengan teknik *string formatting* (menggunakan operator `%`).
2.  **Manajemen Variabel & Objek:** Memahami fleksibilitas *Dynamic Typing* pada Python dan penggunaan fungsi `type()` untuk identifikasi kelas objek.
3.  **Prosedur Casting:** Konversi antar tipe data (Integer, Float, String) untuk menjaga integritas data saat proses komputasi.
4.  **Operasi Logika & Aritmatika:** Penerapan operator matematis dasar (termasuk perpangkatan dan modulo) serta operator perbandingan (`is`, `in`, `and`, `or`).
5.  **Audit Library:** Teknik membedah kapabilitas sebuah modul (seperti `math` dan `time`) menggunakan perintah `dir()` dan `help()`.

## 🔍 Analisis & Kesimpulan Teknis
Setelah melakukan praktik mandiri, berikut adalah beberapa catatan analisis saya:

1.  **Integritas Data Input:** Saya mengidentifikasi bahwa Python memperlakukan setiap input dari pengguna sebagai objek **String**. Oleh karena itu, penggunaan *Casting* atau konversi eksplisit sangat vital agar data numerik dapat diproses dalam fungsi aritmatika tanpa menyebabkan *Type Error*.
2.  **Kesiapan Struktur Data:** Memahami cara kerja variabel dan operator logika merupakan kunci dalam manipulasi data citra. Karena sebuah gambar pada dasarnya adalah matriks angka, maka logika perbandingan dan aritmatika ini akan sangat sering digunakan pada tahap *image processing* di masa mendatang.
3.  **Kesimpulan:** Python menawarkan sintaksis yang sangat efisien untuk pengembangan aplikasi berbasis AI. Penggunaan VS Code sebagai editor sangat membantu dalam mempercepat proses *debugging* dan pengujian logika dasar program melalui terminal yang responsif.
